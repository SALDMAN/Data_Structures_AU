# Data_Structures_AU
This repository containes contect from the course "Data Structures" in Ariel Universtiy at the Computer Science Faculty

The course containes 3 main part's:
1. Calculating complexity of a function
2. Types of Sorting and searching in structures like array's and more.
3. Types of Data Structures like Stacks, Linked lists, Binary trees,Queue, and many more.

This course is a mandatory course in every B.Sc in Computer Science especily in Ariel University.
The course blocks every other programming course in the major especially Algorithms.
